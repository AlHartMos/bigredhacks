Common Mistakes to Watch Out For

  • Forgetting to repeat the key: Make sure the key is repeated or extended to match the length of the plaintext exactly.
  • Wrong letter shifts: Remember that A = 0, B = 1, C = 2, etc. Don’t accidentally start with A = 1!
  • Misalignment: Be careful not to skip or double-use letters when writing out the key and message.
  • Math errors: The letter shift is like modular math — if you go past Z, wrap back around to A.
  • Using the wrong alphabet: Make sure you're only using capital letters A–Z with no spaces or punctuation.
  • Mixing up encoding and decoding: Encoding = shift forward, decoding = shift backward.

Don't worry if you made a mistake — decoding by hand is tricky at first, but practice makes it easier!
